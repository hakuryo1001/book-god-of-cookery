# Fonts for God of Cookery book

Put Jyutcitzi and CJK fonts here so the main book and cover template can display Jyutcitzi (PUA) and Korean.

**Required for Jyutcitzi on cover and in text:**

- `JyutcitziWithSourceHanSerifTCRegular.ttf`
- `JyutcitziWithPMingLiURegular.ttf` (optional, used for some ruby)

**Optional:**

- `batang.ttf` (Korean)
- `BabelStone Han.ttf` (CJK; ctex may use a system font if missing)

After adding the `.ttf` files, rebuild the cover template; Jyutcitzi should then appear in the PDF.
